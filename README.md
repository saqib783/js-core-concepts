Welcome to **JS Core Concepts Frontend Demo**, a professional, interactive, and visually appealing project designed to help learners understand the fundamental JavaScript concepts with live frontend examples.

## ðŸ”¹ Features

This project demonstrates:

1. **Function Types**

   * Function Declaration
   * Function Expression
   * Arrow Function
   * Anonymous Function

2. **Scope**

   * Global Scope
   * Function Scope
   * Block Scope

3. **Closures**

   * Private counters
   * Persistent variables inside functions

4. **High-Order Functions**

   * Functions that take other functions as arguments
   * Functions that return functions

5. **IIFE (Immediately Invoked Function Expressions)**

   * Self-executing functions
   * Encapsulation & private variables

6. **Callback Functions**

   * Functions passed as arguments to execute after a task
   * Demonstrated with delayed messages

7. **Frontend Interactive UI**

   * Smooth animations for dynamic content
   * Gradient animated headings
   * Glassmorphic styled cards
   * Hover effects & smooth transitions
   * All examples visible in a single fixed viewport (no scrolling needed)
